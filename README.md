# FastAPI Starter Project

This is a minimal FastAPI project.

## How to Run

1. Install dependencies:
   
   ```sh
   /Users/shubham/projects/bimakartweb/python/.venv/bin/python -m pip install -r requirements.txt
   ```

2. Start the server:
   
   ```sh
   /Users/shubham/projects/bimakartweb/python/.venv/bin/python -m uvicorn main:app --reload
   ```

3. Visit [http://127.0.0.1:8000](http://127.0.0.1:8000) in your browser.

## Project Structure
- main.py: FastAPI app entry point
- requirements.txt: Python dependencies
